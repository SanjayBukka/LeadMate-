## CodeClarity Agent Features

The CodeClarity agent provides the following capabilities:

- **Repository Analysis**: Analyzes Git repositories to extract commit data and file statistics
- **Developer Insights**: Provides metrics on developer contributions and collaboration patterns
- **Code Quality Recommendations**: Generates insights about code quality based on commit history
- **Team Recommendations**: Offers suggestions for team formation based on code analysis
- **AI Chat Interface**: Allows users to ask questions about repositories and get AI-powered answers
