# Mermaid Diagrams to Create for LeadMate Report

This document lists the 7 most important Mermaid diagrams that should be converted to images for the LeadMate report. Each diagram is identified by its section name and filename that should be used.

## Instructions for Creating Images

1. Copy the Mermaid code for each diagram from the appendix in the LaTeX report
2. Use a Mermaid editor or tool to render the diagram as an image
3. Save each image in the `images` directory with the specified filename
4. Ensure all images are in PNG format with appropriate resolution

## Tools for Converting Mermaid to Images

You can use any of the following tools to convert Mermaid code to images:

1. **Mermaid Live Editor**: https://mermaid.live
2. **VS Code with Mermaid extension**
3. **Online converters** like https://mermaid.ink
4. **Command line tools** like `mermaid-cli`

Simply copy the Mermaid code from the corresponding section in the appendix and paste it into your chosen tool to generate the image.
